HTML5 Features Demo

Audio · Video · Canvas · SVG — with muted, loop, autoplay attributes

1. Audio Tag

<audio>
  • controls
  • loop
  • muted
  • autoplay
  • preload="auto"

muted – starts with audio muted (required for autoplay in most browsers).
loop – replays automatically when the audio ends.
autoplay – starts playing as soon as the page loads (requires muted in Chrome/Firefox).

2. Video Tag

<video>
  • controls
  • loop
  • muted
  • autoplay
  • poster="..."
  • width="100%"

poster – image shown before the video plays.
muted + autoplay – required combination for auto-playing videos in modern browsers.

3. Canvas Tag – Shapes Drawn with JavaScript

<canvas>
  • fillRect()
  • arc() – circle
  • lineTo() – triangle
  • fillText()
  • createLinearGradient()

Canvas shapes are pixel-based and drawn using the 2D rendering context via JavaScript.

4. SVG Tag – Scalable Vector Shapes

<svg>
  • <rect>
  • <circle>
  • <ellipse>
  • <polygon> – triangle
  • <polygon> – star
  • <line>
  • <text>

SVG shapes are resolution-independent vectors defined in XML. Unlike Canvas, they are part of the DOM and can be styled with CSS and manipulated with JavaScript.
